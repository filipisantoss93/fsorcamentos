# Imagens da plataforma

Pasta destinada aos arquivos de imagem do FS Orçamentos.

Sugestão de organização:

- `branding/` — logotipo, ícones e identidade visual
- `efex/` — imagens do personagem Efex
- `pages/` — imagens utilizadas nas páginas
- `uploads/` — arquivos enviados manualmente para uso na plataforma

Use nomes de arquivo em minúsculas, sem espaços e com hífens. Exemplo: `efex-multimetro.png`.
